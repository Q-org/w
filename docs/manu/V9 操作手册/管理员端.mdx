# 管理员端

## 管理员登录

> 打开IE浏览器在地址栏输入：[http://网址: 端口号](http://网址:%20端口号)，显示首页，点击“登录”。跳转到登录页面：登录系统管理员初始帐号admin，（默认密码qianxue.net）。点击登录进入系统。
