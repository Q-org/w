:::info 要求

2019年7月6日，重庆大宇科技有限公司，销售出DMF产品2000千克（单价12元/千克、商品编号01），物流部根据销售合同向第二仓库请求发货。王库管发出商品2000千克。请编制出库单。请根据背景资料，审核出库单，将有错误的栏目，在空白的表单上重新填写。

:::
